# Dashboard Project

## Çakışmayı Çözme(Öğrenci B)
-Conflit i çözmek başta kafa karıştırıcıydı ama modları temizleyip baştan yapınca push yapmak sorunsuz oldu.
-Bu süreçte conflit çözme mantığını kavradım. İşbirliği yaparken nelere dikkat etmem gerektiğini anladım.

## Yeni Kodu Çekme ve Final (Öğrenci A)
- Arkadaşımın çözdüğü conflict’i pull yaptıktan sonra kendi final satırımı eklemek kolay oldu
  ve sistem sorunsuz çalıştı.
-Süreçte bir problem yaşadık: Önce arkadaşımın kendi dosyasında benim son yaptığım değişiklikler görünüyordu,
  bu yüzden push yaparken conflict oluşmadı. 
- Teorik olarak conflict çıkması gerekiyordu ama her iki dosyanın güncel olması nedeniyle Git hata vermedi. 
- Bu durum, conflict senaryosunu deneyimlememizi başlangıçta engelledi ve süreci biraz karışık gösterdi.
-   Ama sonuc olarak adımlarımızı tekrar ederek çözdük.
